## üìä What Is This?

A markdown-first presentation tool integrated into the records platform:
- **Users**: Enterprise teams creating client presentations
- **Pain**: 40% export to external tools (PowerPoint, Keynote)
- **Solution**: Create, present, and share slide decks without context switching
- **Goal**: 20% adoption, 50% export/share rate, 4.5+ rating

### MVP Features (11 total)
1. Presenter display with speaker notes
2. Grid overview (press G)
3. Laser pointer (Cmd/Ctrl + mouse)
4. Black screen mode (press B)
5. PDF export
6. PowerPoint export
7. Share links & embed code
8. Image management (drag-drop)
9. Slide reordering
10. Comments on slides
11. Version history
